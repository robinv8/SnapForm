SnapForm

An intelligent Chrome extension that auto-fills web forms with realistic test data using AI or local heuristics.

Install from Chrome Web Store View on GitHub

One-Click Auto Fill

Click the floating icon on any detected form or use the popup to fill all fields instantly.

AI or Local Mode

Use Gemini, OpenAI, DeepSeek and more, or rely on built-in heuristics with no API key.

Visual Feedback

Watch the form light up and fields pulse as SnapForm fills them in real time.
